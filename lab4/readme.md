# Визуализация алгоритмов растризации

## Описание проекта
Приложение для визуализации и сравнения основных алгоритмов растризации отрезков и кривых, реализованное на Python с использованием Tkinter.

## Реализованные алгоритмы
1. **Пошаговый алгоритм** - базовый алгоритм рисования отрезков
2. **Алгоритм ЦДА (Digital Differential Analyzer)** - алгоритм цифрового дифференциального анализатора
3. **Алгоритм Брезенхема** - целочисленный алгоритм для отрезков
4. **Алгоритм Брезенхема для окружностей** - модификация для рисования окружностей
5. **Алгоритм Кастла-Питвея** - модификация алгоритма Брезенхема
6. **Алгоритм Ву** - алгоритм сглаживания линий (антиалиасинг)
