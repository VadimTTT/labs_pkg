## Описание
GUI приложение для анализа метаданных графических файлов. Поддерживаемые форматы:
- JPEG/JPG
- GIF
- TIFF/TIF
- BMP
- PNG
- PCX

## Возможности
- Анализ одного файла с превью изображения
- Пакетная обработка папок (до 100 000 файлов)
- Отображение основной информации:
  - Размер в пикселях
  - Разрешение (DPI)
  - Глубина цвета
  - Тип сжатия
- Дополнительные метаданные для форматов:
  - JPEG: EXIF данные, качество
  - GIF: количество кадров, палитра
  - TIFF: TIFF теги
  - PNG: параметры сжатия
- Экспорт результатов в CSV
- Статистика обработки

## Запуск

### Готовый EXE файл (Windows)
1. Перейдите в папку \`lab2/\`
2. Запустите \`ImageAnalyzer.exe\`
## Использование

### Режим \"Один файл\"
1. Нажмите \"Выбрать файл\"
2. Выберите изображение
3. Просмотрите метаданные и превью

### Режим \"Папка с файлами\"
1. Нажмите \"Выбрать папку\"
2. Выберите папку с изображениями
3. Нажмите \"Начать обработку\"
4. Дождитесь завершения обработки
5. Просмотрите результаты в таблице

### Экспорт результатов
1. После обработки папки нажмите \"Экспорт в CSV\"
2. Выберите место сохранения файла
3. Получите CSV файл со всеми метаданными

## Производительность
- Обработка одного файла: < 0.1 секунды
- Обработка 600 файлов (~2 ГБ): ~30-60 секунд
- Поддержка многопоточной обработки (не блокирует интерфейс)

## Форматы вывода информации
### Основная информация (для всех форматов):
- Имя файла
- Размер (ширина × высота в пикселях)
- Разрешение (DPI)
- Глубина цвета (бит на пиксель)
- Тип сжатия
- Размер файла

### Дополнительная информация (форматозависимая):
#### JPEG:
- Качество сжатия
- Прогрессивное сжатие
- EXIF данные (производитель камеры, модель, выдержка, диафрагма, ISO и т.д.)

#### GIF:
- Количество кадров (для анимации)
- Количество цветов в палитре
- Длительность кадра

#### TIFF:
- TIFF теги
- Фотометрическая интерпретация
- Битов на выборку

#### PNG:
- Тип сжатия
- Гамма-коррекция

## Тестирование
Приложение было протестировано на:
- Windows 10/11
- Наборе из 600 JPEG файлов (~2 ГБ)
- Файлах различных форматов из архива проверки

## Особенности реализации
- Использует библиотеку Pillow для работы с изображениями
- Многопоточная обработка для папок
- Валидация входных данных
- Обработка ошибок чтения файлов
- Удобный табличный интерфейс для результатов
