# Визуализация алгоритмов отсечения отрезков и многоугольников

## Описание проекта
Приложение для визуализации и сравнения алгоритмов отсечения отрезков и многоугольников, реализованное на Python с использованием Tkinter.

## Реализованные алгоритмы

### Часть 1: Отсечение отрезков прямоугольным окном
1. **Алгоритм Сазерленда-Коэна** - с использованием кодов областей
2. **Алгоритм Лианга-Барски** - параметрический подход
3. **Алгоритм средней точки** - алгоритм "разделяй и властвуй"

### Часть 2: Отсечение отрезков и многоугольников
1. **Алгоритм Cyrus-Beck** - отсечение отрезков выпуклым многоугольником
2. **Алгоритм Вейлера-Азертона** - отсечение выпуклых многоугольников

## Особенности приложения

### Визуализация
- Система координат с осями X и Y
- Масштабируемая сетка с подписями
- Различные цвета для разных типов объектов
- Отображение всех этапов отсечения

### Функциональность
- Загрузка данных из текстового файла
- Ручной ввод координат
- Генерация случайных объектов
- Сохранение результатов в файл
- Измерение времени выполнения алгоритмов

### Интерфейс
- Вкладки для различных типов отсечения
- Информационная панель с вычислениями
- Панель управления с настройками
- Возможность масштабирования
